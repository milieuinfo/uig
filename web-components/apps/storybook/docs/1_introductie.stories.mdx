# Commando's

## Applicaties

`npx nx serve admin-apps` om de app met de landingspagina te starten

`npx nx serve beheer-keuzelijsten` voor het beheer van de keuzelijsten

`npx nx serve beheer-modellen` voor het beheer van de OTL / Subset en OEF modellen

`npx nx serve subset-extern` voor de externe subset tool - deze onder /otltool

`npx nx serve subset-interne` voor de interne subset tool - deze onder /oat


## Storybook

`npx nx storybook storybook` om Storybook op te starten

`npx nx e2e storybook-e2e --watch` om Storybook op te starten en de Cypress test console te openen

`npx nx e2e storybook-e2e` om Storybook op te starten en de Cypress testen eenmalig te laten lopen

`npx nx build-storybook storybook` om een distribueerbare Storybook te bouwen

`npm run docs:json` om compodoc te laten lopen
